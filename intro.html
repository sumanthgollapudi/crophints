In this project, we created an application called "Detect crop hints" that generates a similar cropped image for the given picture. This can be achieved by retrieving an image from the Facebook account of the user using "GraphAPI Tool" and the selected picture will send to Google cloud vision API, Based on the object, facial expressions, text detected in the image by API, it will generate labels. Using those labels this app provides similar images related to the labels.

Java class

Cloud.Java - This class performs the following task:

  1. Invokes Cloud vision API and fetch the images retrieved from Facebook.
  2. Store and retrieve image_ID, image_URL, and the labels to/from the Datastore.

Upload.Java - This servlet is used to create google images search link using label:

  1. All the functionalities are retrieved first and then concatenated.
  2. Then google search link is created and shown as final output.

JSP Files

Home.jsp

  1. Provides an interface to select images either from Facebook or upload from the local file system.
  2. If the user selects an image from Facebook, then pass the selected image will be sent to the servlet for processing.
